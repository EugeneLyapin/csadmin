# csadmin
counter strike 1.6 control panel

## Usage
This Web interface helps to create/delete/launch/stop/modify counter strike servers.

## Prerequisites
* Perl5
* MySQL server and client
* Apache2+PERL5LIB

Perl modules:
* Template
* Sys::SigAction
* CGI
* CGI::Cookie
* Config::Auto
* DBI
* Data::Dumper
* Exporter
* File::Basename
* File::Path
* FindBin
* GD
* HTTP::Request::Common
* LWP::UserAgent
* Net::FTP
* POSIX
* XML::Simple
* IO::File
* LWP
* Storable
